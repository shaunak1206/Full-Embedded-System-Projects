Tester

Readme
 
This project can be used to test the GPIO functionality of the MSPM0G3507 LaunchPad.
 

Derived from Texas Instrument code
  • uart_rw_multibyte_fifo_poll_LP_MSPM0G3507_nortos_ticlang
  • gpio_toggle_output_LP_MSPM0G3507_nortos_ticlang

Hardware Required

  • MSPM0G3507 LaunchPad™ Development Kit (LP-MSPM0G3507)
  • LaunchPad Tester boosterpack

Pin assignments
  • Serial TxD: PA10 is UART0 Tx (MSPM0 to PC) index 20 in IOMUX PINCM table
  • Serial RxD: PA11 is UART0 Rx (PC to MSPM0) index 21 in IOMUX PINCM table
  • Output: PA0 is red LED1, index 0 in IOMUX PINCM table
LaunchPad Jumpers
  • Remove J16 J17 J18: Disconnects light sensor from the microcontroller PA22 PA27 PA26
  • J9: Remove J9: Disconnects thermistor from PB24
  • J14: Connect PB23 to SW1
  • J15: Connect PA16 to SW2
  • J4: Connects PA0 to red LED
  • J21: Connects PA10 to XDS_UART
  • J22: Connects PA11 to XDS_UART
